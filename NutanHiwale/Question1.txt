What is cherry-pick? Explain the use case of it also write steps to do it?
cherry-pick: Cherry-picking in Git stands for applying some commit from one
	     branch into another branch.
Use Case: Accidently make a commit in a wrong branch.
	  Suppose I want to make a commit in the master branch, but by
   	  mistake we made it in any other branch. First add file then 
	  commit changes.
